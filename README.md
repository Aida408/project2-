# # Веб-приложение для классификации изображений CIFAR-10

## 1. Обзор проекта
Данный проект представляет собой end-to-end решение для классификации изображений из набора данных CIFAR-10. Он включает обучение сверточной нейронной сети (CNN) на PyTorch и создание интерактивного веб-приложения на Streamlit, позволяющего пользователю загружать изображения и получать предсказания в реальном времени. 

**Бизнес-контекст / Применение:**  
Классификация изображений — фундаментальная задача компьютерного зрения, которая применяется в системах безопасности, медицинской диагностике, автономных транспортных средствах и пользовательских приложениях (например, распознавание объектов на фото). Проект демонстрирует полный процесс разработки таких систем — от подготовки данных до развертывания модели.

## 2. Описание задачи
Проект состоит из двух частей:

- **Backend (PyTorch):** обучение CNN для классификации изображений на 10 классов.  
- **Frontend (Streamlit):** веб-приложение, позволяющее пользователю загружать изображения и получать предсказанный класс.

## 3. Набор данных
**Источник:** [CIFAR-10 Dataset](https://www.cs.toronto.edu/~kriz/cifar.html)  
**Описание:** 60 000 цветных изображений размером 32x32 пикселя, разделённых на 10 классов, по 6 000 изображений в каждом классе.  
**Классы:** `['plane', 'car', 'bird', 'cat', 'deer', 'dog', 'frog', 'horse', 'ship', 'truck']`  

## 4. Основные этапы проекта
### Часть 1: Модель на PyTorch
1. **Подготовка данных:**  
   - Загрузка CIFAR-10 через `torchvision.datasets.CIFAR10`.  
   - Преобразование изображений: нормализация, конвертация в тензоры.  
   - Создание `DataLoader` для обучения и тестирования.
2. **Проектирование и обучение CNN:**  
   - Определение архитектуры модели (`torch.nn.Module`).  
   - Выбор функции потерь (`nn.CrossEntropyLoss`) и оптимизатора (`SGD` или `Adam`).  
   - Обучение модели на тренировочном наборе.
3. **Оценка и сохранение модели:**  
   - Проверка точности на тестовом наборе.  
   - Сохранение весов модели: `torch.save(net.state_dict(), './cifar_net.pth')`.

### Часть 2: Веб-приложение на Streamlit
1. **Интерфейс:**  
   - Загрузка изображения через `st.file_uploader`.  
   - Отображение загруженного изображения.
2. **Интеграция модели:**  
   - Загрузка сохраненной модели.  
   - Применение тех же преобразований, что и при обучении.  
   - Предсказание класса и вывод результата.
